# পেমেন্ট কনফার্মেশন - Payment Confirmation

একটি আধুনিক এবং দ্রুত পেমেন্ট কনফার্মেশন ওয়েবসাইট। এটি ব্যবহারকারীদের পেমেন্ট তথ্য ইনপুট করতে এবং সাফল্যের নিশ্চয়তা পেতে দেয়।

## ফিচার (Features)

✅ **বাংলা-ইংরেজি মিশ্রিত ইন্টারফেস**
✅ **সুন্দর গ্রিন-পিংক ডিজাইন**
✅ **মোবাইল রেসপন্সিভ**
✅ **অ্যানিমেটেড চেকমার্ক**
✅ **ডেটা ভ্যালিডেশন**
✅ **তারিখ ও সময় সাপোর্ট**

## ক্ষেত্র (Fields)

1. **প্রেরক (Sender)** - আপনার নাম
2. **পরিমান (Amount)** - টাকার পরিমান
3. **ট্রানজেকশন (Transaction ID)** - লেনদেন আইডি
4. **তারিখ ও সময় (Date & Time)** - সময়ের তথ্য

## ব্যবহার করুন (Usage)

1. সব ফিল্ড পূরণ করুন
2. "সাবমিট করুন" বাটনে ক্লিক করুন
3. সাফল্যের পেজ দেখুন
4. "নতুন পেমেন্ট" বাটনে ক্লিক করে নতুন লেনদেন করুন
